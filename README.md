Bus Boss
===============================================================================
A quick visualization of some live NYC Bus data


Details
===============================================================================
* Hosted on local machine
* used virtualenv to setup minimal environment
* custom Bootstrap css hacks to center navbar
* see useful docs and tech stack, below
